📱 Mobile / Tablet Edition This is a standalone web app — no browser extension needed. Open it in Chrome or Safari, then add to Home Screen for the best experience. Actions (key presses, clicks) are simulated on elements in this same page/tab.
⌨ KEY PRESS MODE
Tap the key box to enter a key.
On mobile, type the key in the input field shown.
Common keys: 1 2 Space F1 q

Press — dispatches keydown + keyup once per interval.
Hold — dispatches keydown, waits N ms, then keyup.

🎯 ACTION MODES
Key Press — dispatch key event once
Key Hold — dispatch keydown, hold N ms, keyup
Left Click — MouseEvent click at XY or selector
Right Click — contextmenu event
Double Click — dblclick event
XY Coords — click at exact pixel coordinates
📐 XY COORDINATES
Enter X and Y pixel coordinates manually.
To find them: open browser DevTools → Console,
then run: document.addEventListener('click', e => console.log(e.clientX, e.clientY))
and tap the target element.
💾 SAVE / LOAD
Profiles auto-save to your browser's localStorage.
They persist across sessions on the same device/browser.
